# Bank Statement Converter

Local app chuyen so phu ngan hang sang file `ERP_Import.xlsx` theo template ERP.

## Cach chay

```powershell
cd "C:\Users\admin\OneDrive\Documents\New project\bank-converter-app"
npm start
```

Mo trinh duyet tai:

```text
http://localhost:5177
```

## Ngan hang dang ho tro

- `BIDV`: sheet `BIDV`, du lieu tu hang 13.
- `TECH`: sheet `TECH`, du lieu tu hang 22.

Co the upload nhieu file cung luc. App se gop tat ca thanh mot file output `ERP_Import_YYYYMMDD_HHMMSS.xlsx`.
